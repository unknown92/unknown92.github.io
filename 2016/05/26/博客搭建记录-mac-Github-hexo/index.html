博客搭建记录-mac+Github+hexo

前言

        鉴于懒癌已经晚期，必须壮士断腕治上一治了。建个博客，大概会是个日记加笔记的混合体，而且还是个技术与非技术（护肤搭配买买买~(≧▽≦)/~啦啦啦）的混合体。第一篇，当然是记录博客搭建过程啦，废话结束，请看下文。

Github

        GitHub是一个利用Git进行版本控制、专门用于存放软件代码与内容的共享虚拟主机服务。GitHub目前已经成为了世界上最大的代码存放网站和开源社区。全球最大的同性交友平台，反正起码百分之八十是程序员，所以…（大误）
http://github.com/ 注册个账号就行

Github Pages

        github提供的服务，可以简单的认为是托管在github上的静态网页，用户可以给项目建立介绍网点，当然，也可以用来搭建博客。
https://pages.github.com/

Websites for you and your projects.
Hosted directly from your GitHub repository. Just edit, push, and your changes are live.

  • 轻量级的博客系统，没有麻烦的配置
  • 使用标记语言，比如Markdown，无需自己搭建服务器
  • 根据Github的限制，对应的每个站有300MB空间
  • 可以绑定自己的域名

        创建个仓库，顾名思义，给你放东西用的，create repository就行，下面的步骤看了，然并卵。

Git

        Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。
        如果你的mac装了XCode，那git就不需要再装了，没装的话，git另外装一下咯。

Node.js

        Hexo是基于Node.js的，所以必须要装这个。

Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine. Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient. Node.js’ package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

https://nodejs.org/en/ 下载，安装，over

Sublime

代码编辑器，好用，over

Hexo

接下来就开始安装Hexo了。（用的命令行，可能中途会出现卡了之类的情况，stop and restart，或者小黑板关掉重新打开）

  1. 输入以下命令
    npm install -g hexo

  2. 建立一个目录，名字任意，我选的是根目录，指向此目录(命令：cd path)，然后执行以下命令
    hexo init npm install

  3. Hexo会自动建立文件，到这里，博客的初步搭建已经完成。

##使用

  • hexo g generate
  • hexo s server，http://localhost:4000/，可以预览
  • hexo d deploy,发布
  • theme：NextT，有中文使用文档，里面有hexo的一些配置教程
