博客搭建记录-mac+Github+hexo

发表于 2016-05-26   |   分类于 blog

前言

        鉴于懒癌已经晚期，必须壮士断腕治上一治了。建个博客，大概会是个日记加笔记的混合体，可能还会混入比如吐槽之类的不明物体。。。第一篇，当然是记录博客搭建过程啦，废话结束，请看下文。

        博客系统有很多，有静态的有动态的，有免费也有收费，想做了解的可以看一下知乎的这篇帖FarBox、Jekyll、Octopress、Hexo…这些博客程序有什么特点？

Github

        欢迎来到全球最大的同性交友平台，开玩笑的😂
        Github是一个利用Git进行版本控制、专门用于存放软件代码与内容的共享虚拟主机服务。GitHub目前已经成为了世界上最大的代码存放网站和开源社区。包括Google、Facebook、Twitter、阿里巴巴、网易等公司都在github开源项目，造福人类啊，感激。

步骤

注册个账号就能使用，注册过程，略。

Github Pages

Websites for you and your projects.
Hosted directly from your GitHub repository. Just edit, push, and your changes are live.

        github提供的服务，可以简单的认为是托管在github上的静态网页，用户可以给项目建立介绍网点，当然，也可以用来搭建博客。

  • 轻量级，配置简单
  • 使用标记语言，比如Markdown，无需自己搭建服务器
  • 根据Github的限制，对应的每个站有300MB空间
  • 可以绑定自己的域名

步骤

        创建个仓库，顾名思义，给你放东西用的，create repository就行，官网中其他的步骤看了一下，然并卵。

Git

        Git是一款免费、开源的分布式版本控制系统，用于敏捷高效地处理任何或小或大的项目。

步骤

        如果你的mac装了XCode，那git就不需要再装了，没装的话，git另外装一下咯。

Node.js

        Hexo是基于Node.js的，所以必须要装这个。

Node.js® is a JavaScript runtime built on Chrome’s V8 JavaScript engine.
Node.js uses an event-driven, non-blocking I/O model that makes it lightweight and efficient.
Node.js’ package ecosystem, npm, is the largest ecosystem of open source libraries in the world.

步骤

下载，安装，over

Sublime

Sublime Text is a sophisticated text editor for code, markup and prose.
You’ll love the slick user interface, extraordinary features and amazing performance.

代码编辑器，好用，over

Hexo

快速、简洁且高效的博客框架

  • 超快速度
    Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染。

  • 支持 Markdown
    Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。

  • 一键部署
    只需一条指令即可部署到 GitHub Pages, Heroku 或其他网站。

  • 丰富的插件
    Hexo 拥有强大的插件系统，安装插件可以让 Hexo 支持 Jade, CoffeeScript。

步骤

        终于到了安装Hexo的时候了。（用的命令行，可能中途会出现卡了之类的情况，stop and restart，或者小黑板关掉重新打开）

  1. 输入以下命令
    npm install -g hexo

  2. 建立一个目录，名字任意，我选的是根目录，指向此目录(命令：cd path)，然后执行以下命令
    hexo init
    npm install

  3. Hexo会自动建立文件，到这里，博客的初步搭建已经完成。

使用

  • hexo g —— generate，
  • hexo s —— server，预览
  • hexo d —— deploy，发布
  • theme：NextT，github上开源的hexo的主题，有中文使用文档，里面有hexo的一些配置教程
